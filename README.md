# Business Buying Hub
Business Buying Hub is a smart B2B shopper profile dashboard that combines analytics, approvals, and cost optimization in one place.

## Features
- Analytics dashboard
- Purchase approvals workflow
- Cost optimization tools
- User-friendly interface
## Installation
1. Clone the repository
2. Open `index.html` in your browser
## Usage
Open the dashboard and explore the analytics and purchase management features.
## Folder Structure
- `css/` – Stylesheets
- `js/` – JavaScript files
- `index.html` – Main page
